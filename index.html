Projects

A collection of various developement projects:

Beacon

This was my final year project. A single page application created in React using JavaScript, Python, Flask, Firebase and Google's Natural Language. The aim was to help students with issues that may arrise during their time at University. The project entailed createing a webpage that allowed users to access and use the system, as well as having a intelligent assistent created in Python using machine learning.

Cloud Calculator

This involed creating a calculator and hosting on the cloud. This calculator was programmed in Java, Python, c# and Golang. as part of the requirements were to have it in four different languages. This also included Error handling, Dockerfiles, JSON Reponces and Ci pipelines. A custom proxy router was also created as well as a frontend service failure handler.

Cloud Search Engine

This project was to create a search engine, which included implemeting a spider indexer, search funtion and a database using postgreSQL. Upon getting these to work locally they were then depoloyed using CI pipelines to google cloud, these also included DockerFiles, and deployement.yaml. Adds were also implementsed via a database using key words that are releated to what the user has searched. This also had monitoring using grafana.
